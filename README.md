# Parkly
### Сайтът е активен на https://parkly-app1.vercel.app/
SmartPark е умно приложение за паркиране, което в реално време показва свободните места на паркинги и улици. С помощта на интелигентни алгоритми и актуални данни, Parkly ти помага бързо да намериш място, да спестиш време, гориво и излишно обикаляне из града.
